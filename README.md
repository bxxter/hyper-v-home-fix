# Enable Hyper-V en Windows Home

Este repositorio contiene un **script `.bat`** que permite **habilitar Hyper-V en Windows 11 Home**, donde esta característica no viene activada por defecto.

El script utiliza **herramientas oficiales de Windows (DISM)** y **no instala software externo**.

Este script:
- Agrega los paquetes necesarios de Hyper-V
- Habilita la característica **Microsoft-Hyper-V-All**
